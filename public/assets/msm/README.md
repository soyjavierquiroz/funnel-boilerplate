# MSM Assets

Estos assets pertenecen al baseline historico Aprender Motores / Maneja Sin Miedo.

Se conservan como referencia temporal mientras el repo padre termina de aislar la configuracion por sitio. No deben usarse como identidad final de otros clones.

Cada clon debe crear su propia carpeta:

```text
public/assets/<site-slug>/
```

Luego debe referenciar esas rutas desde `src/site/**`, normalmente desde el DNA activo.
